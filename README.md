# Liri-Node-App

## ABOUT
LIRI is a Language Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives back data. Different commands that works with this app are:

   * `concert-this`

   * `spotify-this-song`

   * `movie-this`

   * `do-what-it-says`
